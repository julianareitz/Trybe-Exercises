Structured Query Language / Linguagem de Consulta Estruturada

- MYSQL COMPARITION
|----------------------------------------|----------------------------------------------|
|         MYSQL                          |                  NoSQL                       |
|----------------------------------------|----------------------------------------------|
|   Organized and esctructured data      |          Not organized data                  |
|----------------------------------------|----------------------------------------------|
|     Secure data                        |        Not that secure data                  |
|----------------------------------------|----------------------------------------------|
|  Multiple Requirements, once at time   |      Multiple access at the same time        |
|----------------------------------------|----------------------------------------------|
|   Related data by constraints          |                                              |
|----------------------------------------|----------------------------------------------|
|              attribuits                |                 attribuits                   |
|----------------------------------------|----------------------------------------------|
|              registers                 |                  documents                   |
|----------------------------------------|----------------------------------------------|
|                 table                  |                collection                    |
|----------------------------------------|----------------------------------------------|

- DATABASE OPERATIONS

| C | Create
| R | Read
| U | Update
| D | Delete


- MYSQL COMMANDS

  (*) SELECT
  (*) WHERE
  (*) UPDATE
  (*) DELETE
  (*) INSERT 

Good pratice -> use it in upper case.


- COMMUN CONSTRAINTS

|-------------|-------------------------------------------------------------------------|
|   NOT NULL  | it didnt accept null values input                                       |
|-------------|-------------------------------------------------------------------------|
|    UNIQUE   | accepts only different values at same column                            |
|-------------|-------------------------------------------------------------------------|
| PRIMARY KEY | unique id of the table : brings NOT NULL && UNIQUE as default           |
|-------------|-------------------------------------------------------------------------|
| FOREIGN KEY | reffers to another table PRIMARY KEY                                    |
|-------------|-------------------------------------------------------------------------|
|   DEFAULT   | registers some value as default in null input cases                     |
|-------------|-------------------------------------------------------------------------|
