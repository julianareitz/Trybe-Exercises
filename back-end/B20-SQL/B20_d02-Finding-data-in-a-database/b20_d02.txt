

QUERIES:

|---------------------------------------------------------------------------|
|                                                                           |
|                     DDL - Data Definition Language                        |
|                                                                           |
|----------|----------------------------------------------------------------|
| CREATE   | Create database, tables, index, views, procedures,             |
|          |    functions and triggers.                                     |
|----------|----------------------------------------------------------------|
| ALTER    |  Alter any estructure of any object.                           |
|----------|----------------------------------------------------------------|
| DROP     |  Delete objects.                                               |
|----------|----------------------------------------------------------------|
| TRUNCATE |  Empty table data with no interaction to database.             |
|----------|----------------------------------------------------------------|


|---------------------------------------------------------------------------|
|                                                                           |
|                     DML: Data Manipulation Language                       |
|                                                                           |
|----------|----------------------------------------------------------------|
| SELECT   | Search for data in a especific database.                       |
|----------|----------------------------------------------------------------|
| INSERT   |  Insert table data.                                            |
|----------|----------------------------------------------------------------|
| UPDATE   |  Updates table data.                                           |
|----------|----------------------------------------------------------------|
| DELETE   |  Delete table data.                                            |
|----------|----------------------------------------------------------------|


|---------------------------------------------------------------------------|
|                                                                           |
|                     DCL: Data Control Language                            |
|                                                                           |
|----------|----------------------------------------------------------------|
| GRANT    | Grant user access                                              |
|----------|----------------------------------------------------------------|
| REVOKE   |  Revokesuser access by GRANT command.                          |
|----------|----------------------------------------------------------------|


|---------------------------------------------------------------------------|
|                                                                           |
|                     TCL: Transactional Control Language                   |
|                                                                           |
|------------|--------------------------------------------------------------|
| COMMIT     | Commit changes to database                                   |
|------------|--------------------------------------------------------------|
| ROLLBACK   |  Rollbacks the command you've typed.                         |
|------------|--------------------------------------------------------------|
|SAVEPOINT   |  Works like a checkpoint, interacting with ROLLBACK          |
|------------|--------------------------------------------------------------|
|TRANSACTION |  Defines when, how, and the transaction scope                |
|------------|--------------------------------------------------------------|


DISTINCT
